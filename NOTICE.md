# Notices

이 저장소는 Seorilabs의 AppsInToss 교육형 미니앱 `원소 주기율표` 소스 코드를 포함합니다.

## Data

- 원소 속성 데이터: PubChem Periodic Table CSV
  - URL: https://pubchem.ncbi.nlm.nih.gov/rest/pug/periodictable/CSV
  - Retrieved: 2026-05-22
  - 사용 항목: 원자량, 표준 상태, 전자배치, 전기음성도, 밀도, 발견 연도
- 한국어 원소명: 대한화학회 제정 원소 이름
  - URL: https://kchem.org/iupacname
  - Retrieved: 2026-05-22
- 주기율표 표시 위치, 분류 색상, 짧은 학습 설명은 앱 UX에 맞춰 Seorilabs가 정리했습니다.
